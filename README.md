# Módulo V: Estrategia Tecnológica + Mindset Digital

Curso diseñado para líderes del sector educativo que necesitan entender el rol estratégico de la tecnología y desarrollar el mindset necesario para liderar la transformación digital de sus instituciones.

## Bloques temáticos

1. **Hello World** — El rol de la tecnología en el negocio educativo
2. **Estrategia Tecnológica** — Definiciones clave y componentes de una estrategia integral
3. **Arquitectura IT Educativa** — Ecosistema de plataformas y ciclo de vida del estudiante
4. **Transformación Digital** — Cultura + Proceso + Tecnología. Framework de adopción de IA
5. **Mindset Digital** — Los 7 pilares que hacen posible la transformación
6. **Challenge** — Diseño de una hoja de ruta tecnológica para una institución real
